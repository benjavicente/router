---
title: How to Install TanStack Router
---

## Prerequisites

- React 18.x.x or 19.x.x
- ReactDOM 18.x.x or 19.x.x with `createRoot` support
- TypeScript 5.3.x or higher (recommended)

## Installation Steps

1. **Install the package**

   Choose your package manager:

   ```sh
   npm install @benjavicente/react-router
   ```

   ```sh
   pnpm add @benjavicente/react-router
   ```

   ```sh
   yarn add @benjavicente/react-router
   ```

   ```sh
   bun add @benjavicente/react-router
   ```

   ```sh
   deno add npm:@benjavicente/react-router
   ```

2. **Verify installation**

   Check that the package appears in your `package.json`:

   ```json
   {
     "dependencies": {
       "@benjavicente/react-router": "^x.x.x"
     }
   }
   ```
